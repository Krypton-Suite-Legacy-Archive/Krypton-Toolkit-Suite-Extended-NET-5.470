Gets or sets a mask indicating which markers this margin can display.

Return Value

Type: UInt32
An unsigned 32-bit value with each bit cooresponding to one of the 32 zero-based Margin indexes. The default is 0x1FFFFFF, which is every marker except folder markers (i.e. 0 through 24).
Remarks
For example, the mask for marker index 10 is 1 shifted left 10 times (1 << 10). MaskFolders is a useful constant for working with just folder margin indexes.
